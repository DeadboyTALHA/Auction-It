## Project Description
A full-stack online auction platform built with MERN stack where users can bid on items and sellers can list products with time-based auctions.

## Tech Stack
- MongoDB - Database
- Express.js - Backend Framework
- React.js - Frontend Library
- Node.js - Runtime Environment
- Socket.io - Real-time bidding
- Firebase - Notifications
- Stripe - Payment Processing
